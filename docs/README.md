# KyberRock Docs

Documentacao tecnica e operacional do projeto KyberRock.

## Estrutura

- `phase-0/`: preparacao, spikes e validacoes tecnicas iniciais.
- `phase-1/`: design tecnico, modelo de dados, contratos e estrategias de sincronizacao.
- `ARCHITECTURE.md`: visao tecnica consolidada do sistema.

## Regra De Segredos

Credenciais reais do OMIE, Firebase, e-mail, impressoras, servidores ou qualquer outro servico externo nao devem ser commitadas. Use arquivos locais ignorados pelo Git, variaveis de ambiente ou cofre de credenciais do sistema operacional.
